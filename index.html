Метод Що вертає Опис Приклад

Різниця між map() & forEach()

  • forEach завжди вертає undefined.Через це його не використаєш одночасно з іншими методами
  • forEach() - Якшо потрібно зробити лише дії з кожним елементом
    map() - Якшо потрібно отримати новий масив на основі вихідних даних

Sort()

  • Сортування чисел по зростанню - sort((a, b) => a - b)

Копіювання масиву

  • clone = [...origin]
  • clone = Array.from(origin)
  • clone = origin.slice()
  • clone = origin.concat()
  • clone = origin.map(el => el)- Самий повільний спосіб!
